## Airbnb clone

![fon](https://www.spinxdigital.com/app/uploads/2022/11/image-airbnb.jpg)


### Open the project yourself!

```bash
git clone https://github.com/Rofiyev/airbnb.git

cd airbnb

npm install

npm run dev

```
